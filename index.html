Step 1 Not started

Telegram

Connect your Telegram bot. You need a bot token from BotFather and, optionally, your numeric user ID so you’re allowed to approve actions.

How do I get these?

Get your bot token

  1. In Telegram, open a chat with @BotFather.
  2. Send /newbot and follow the prompts (name + username ending in bot).
  3. BotFather replies with a token like 123456789:AAH… — paste it above.

Get your user ID

  1. In Telegram, open a chat with @userinfobot (or @RawDataBot).
  2. Press Start — it replies with your numeric Id (e.g. 123456789).
  3. Paste that number above. Add several, comma-separated, to allow more people.
openclaw channels add --channel telegram --token <BOT_TOKEN>

Tip: the client must open the bot in Telegram and press Start before it can receive the confirmation message.

Step 4 Not started

    
Start / restart gateway

    
Apply your configuration. On first run this installs and starts the gateway service; after that it just restarts it.

    
openclaw config set gateway.mode local
openclaw gateway install --force
openclaw gateway restart
